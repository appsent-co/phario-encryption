# @appsent-co/phario-encryption

A encryption library for the Phario app

## Installation

```sh
npm install @appsent-co/phario-encryption
```

## Usage

```js
import { multiply } from "@appsent-co/phario-encryption";

// ...

const result = await multiply(3, 7);
```
